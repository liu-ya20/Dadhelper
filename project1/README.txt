
process.py使用方法
*************************************************************
1. 确认process.py文件的存储路径，下面以D:\数据处理小程序为例
2. win+R呼出搜索框输入cmd打开终端
3. 输入下面指令切换到process.py所在文件夹

cd /d D:/数据处理小程序

4. 输入下面指令运行process.py

python process.py

5. 前往result文件夹找到输出文件
*************************************************************

注意！
*************************************************************
1. 每一次运行会覆盖掉之前的名字为result文件，除非更改代码第8行的文件名
2. 输出结果中合约电价放在了最后一列，其中数据已经通过程序将相关的两项相乘计算得到，但没有设置此列
   计算逻辑，如果需要设置需要到Excel软件里手动设置此列的计算逻辑


示例数据以及结果分别放在data和result文件夹中，template文件夹中为参考模板文件，不影响代码运行
